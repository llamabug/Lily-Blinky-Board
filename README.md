# Lily-Blinky-Board
My first hardware! A blinky board in the custom shape of a lily (my favourite flower!)  🪷

![3D board image](image.png)

This project features 10 LED lights that blink in sequence, and was made in KICAD from a Hack Club Stasis tutorial.
<br> It also uses a NE555P timer, a 4017, a Conn_01x02Socket and multiple capacitators/resistors (I believe there is a kit for these?). The custom PCB costs $2.30, with $1 shipping from seeed.
<br>As my first hardware project, it was pretty difficult, especially with the DXF files, but I am really proud of the final result, and it was pretty fun to make, so I will almost certainly be building more hardware in the future.
